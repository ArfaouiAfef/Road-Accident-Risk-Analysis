# 🚗 Road Accident Risk Analysis – Power BI Dashboard & Case Study

An interactive Power BI dashboard and complete data analytics case study based on the Kaggle Playground Series S5E10 dataset. This project focuses on analyzing road accident risk factors using data cleaning, DAX measures, KPIs, and visual analytics.

---

## 📊 Project Overview

The objective of this project is to explore and analyze road accident data to identify the key factors influencing accident risk. The dashboard provides interactive visualizations, KPI tracking, and business insights to support data-driven decision-making.

---

## 🛠 Tools & Technologies

- Power BI
- Power Query
- DAX
- Kaggle Dataset
- Canva

---

## 📈 Dashboard Features

### 🔹 KPI Cards
- Total Incidents
- High-Risk Incident Percentage
- Average Accident Risk
- Driver Risk Score
- Average Vehicle Age

### 🔹 General Analysis
- Accident distribution by risk category
- Incidents by road type
- Incidents by weather condition

### 🔹 Driver Analysis
- Speed vs Accident Risk
- Driver experience analysis
- Reaction time analysis

### 🔹 Interactive Dashboard
- Dynamic slicers and filters
- Interactive visuals
- User-friendly dashboard design

---

## 🎯 Key Insights

- High vehicle speed significantly increases accident risk
- Inexperienced drivers are more exposed to accidents
- Highway roads show the highest incident levels
- Extreme weather conditions increase accident probability
